# Snake Game

React + Vite implementation of the classic Snake game.

## Run locally

```bash
npm install
npm run dev
```

The high score is saved by the Vite dev server in:

```text
data/highscore.txt
```

## Structure

```text
data/
  highscore.txt
src/
  App.jsx
  main.jsx
  components/
    Board.jsx
    Food.jsx
    HighScore.jsx
    Score.jsx
    Snake.jsx
  styles/
    game.css
```
